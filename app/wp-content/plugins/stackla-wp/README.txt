== Installation ==

1. Upload `stackla-wp.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click the **Stackla** tab that appears in the dashboard menu
1. Enter your stack name, client id & client secret in the fields provided and click *save*
1. Select which post types you would like the Stackla custom metabox to appear on
1. Once you have successfully saved these fields, click the *authorise* button to authorise your WordPress user account with Stackla
1. Note: each WordPress user will need to authorise themselves with Stackla separately in order to make changes to a Stackla custom metabox

= How do I get my stack name, client id & client secret =

You'll need a registered Stackla account for these, for more information visit http://stackla.com/platform/plugins/#wordpress

= How do I change the Network for a Term? =

In order to do this, you will need to remove the term first and then create a new one

= How do I change the Type for a Term? =

Again, in order to do this, first remove the term, then create a new one with the correct type
